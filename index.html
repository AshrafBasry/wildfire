What is Xenlism WildFire?

xenlism is Computer Graphic And Programming project to make something batter.

xenlism is about minimalism and realism.

xenlism is not material design and never.

xenlism wildfire is icon theme for *nix desktop environment.

inspired by Nokia's meego and Apple iOS icon.

best for gnome and (Maybe) unity,mate,cinamon.

xenlism wildfire all icon is vector graphics (svg,ai).

xenlism wildfire source code included illustrator file (ai) and Prototype Template icons (svg).

prototype template icon is easy script to build many of number different color by color scheme file.

The Wildfire Script included script for convert svg,ai to any Other icon format ai-svg,ai-icns,ai-png,svg-icns,svg-svg and svg-png.

The Wildfire Script work on Mac OS X , BSD and Linux.

Get Installations

You can install Xenlism WildFire by pacman and Apt-Get Git or WGET and CURl

We have many solution to help you install Xenlism WildFire to your system by easy.

Debian/Ubuntu Apt-Get Repository


Xenlism Wildfire support Debian/Ubuntu Base apt-get Repositories.

Following Command line in Terminal. 


	sudo nano /etc/apt/source.list

add "deb http://repo-xen.rhcloud.com deb/" (without qoute) to /etc/apt/source.list. 


	deb http://repo-xen.rhcloud.com deb/

Update System and Install Xenlism Wildfire icon theme And Xenlism Artwork Wallpapers. 


	sudo apt-get update
	sudo apt-get install xenlism-wildfire-icon-theme
	sudo apt-get install xenlism-artwork-wallpapers

Arch Linux Pacman Repository


Xenlism Wildfire support Arch Linux Base apt-get Repositories.

Following Command line in Terminal. 


		sudo nano /etc/pacman.conf

add config section to /etc/pacman.conf 


	[xenlism-arch]
	Server = http://repo-xen.rhcloud.com/arch
	SigLevel = Never

Update System and Install Xenlism Wildfire icon theme And Xenlism Artwork Wallpapers. 


	sudo pacman -Syyu
	sudo pacman -S xenlism-wildfire

Other Linux Distributions

Xenlism Wildfire Installations Support A most Linux and *nix System with CURl or WGET and GIT.


Install with wget


	bash <(wget -qO- https://raw.githubusercontent.com/xenlism/wildfire/master/INSTALL/online.install)

Install with curl


	bash <(curl -s https://raw.githubusercontent.com/xenlism/wildfire/master/INSTALL/online.install)

Install with git


	cd /tmp;git clone https://github.com/xenlism/wildfire.git;bash /tmp/wildfire/INSTALL/offine.install;

Install with wget and git


	bash <(wget -qO- https://raw.githubusercontent.com/xenlism/wildfire/master/INSTALL/git.install)

Install with curl and git


	bash <(curl -s https://raw.githubusercontent.com/xenlism/wildfire/master/INSTALL/git.install)
